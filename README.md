Java商城秒杀系统<br/>
技术列表:<br/>
SpringBoot(整个系统的奠基)<br/>
SpringMVC+MyBatis+Jsp(用于构建一个完整的系统)
分布式锁的实现与应用(解决高并发多线程对共享资源并发访问带来的安全问题)<br/>
Redis(缓存中间件，用于数据的缓存与分布式锁的实现)<br/>
ZooKeeper(注册中心，用于分布式锁的实现)<br/>
RabbitMQ(消息中间件，用于业务模块异步通信与接口限流)<br/>
Mysql(商品详情、订单记录、秒杀处理的优化)<br/>
雪花算法(分布式唯一ID的生成方式，用于高效生成唯一编码的算法)<br/>
SpringBoot邮件服务、Lamda表达式、Jmeter压力测试等<br/>
